# 骑行伙伴共享实时位置插件 PRD

## 产品定位

高德骑行导航内的多人同行安全插件。它不是新的导航产品，而是在现有骑行导航页面中增加“同行伙伴实时可见”的安全感知层。

## 用户问题

多人骑行时，领骑和跟骑经常需要回头确认队友位置。这会带来：

- 注意力离开前方路况
- 队伍被红绿灯或路口切断后难以及时判断
- 掉队伙伴不容易被发现
- 临时停靠点沟通成本高

## 核心用户故事

- 作为领骑，我希望在高德骑行导航里看到队友是否掉队，避免频繁回头。
- 作为跟骑，我希望知道自己离领骑还有多远，以及是否还在正确路线上。
- 作为收队，我希望快速判断谁落在最后，是否需要提醒全队等待。

## MVP 功能

1. 创建共享骑行房间
2. 微信邀请同行骑友加入
3. 导航地图上显示队友头像点
4. 按骑行路线里程计算距离
5. 自动判断智能领骑、跟骑、收队
6. 掉队风险提醒
7. 一键退出共享

## 非目标

- 不做完整社交聊天
- 不保存长期历史轨迹
- 不替代高德原生导航
- 不做公开陌生人组队

## 成功指标

- 骑行中回头确认次数下降
- 掉队发现时间缩短
- 用户愿意在下一次多人骑行继续开启
- 导航页遮挡投诉低

## 风险

- 高德地图 App 暂无公开第三方插件分发机制
- 微信内置浏览器可能限制调起高德 App
- 真实骑行中 GPS 漂移会影响距离判断
- 骑行时 UI 必须极简，不能诱导长时间看屏
